
# TracerTNG - Kali OSINT Tool scans automated

Più informazioni su  requisiti e funzionamento su requirements.txt allegato con script

## Il tool TracerTNG è strettamente per automazione di operazioni di OSINT riguardanati l'ambito del penetration testing ed è da usare esclusivamente con permissione esplicità di tutte le parti.

## Tool utilizzati (spesso inclusi con kali):
- nslookup
- whois
- sublist3r
- subfinder
- theHarvester
- recon-ng

## API utilizzate (opzionali):
- Gemini API con seguente formato: GEMINI_API_KEY=your_gemini_key
- Shodan API con seguente formato: SHODAN_API_KEY=your_shodan_key

## Come installare requisiti
in CLI Kali (o qualsiasi macchina con pip):
pip install -r requirements.txt

## DISCLAIMER
Non sono programmatore, non conosco bene python, questo script è estremamente non-convenzionale e creato solo come esercizio per imparare a integrare tool esterni ed API per il mio corso durante le lezioni di OSINT e di strumenti ad essa correlati.
